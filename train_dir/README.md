## Train directory 

Use this directory to train machine learning models based on folders of files.

Files can take form:

| File type | extension | 
| ------------- |-------------| 
| audio file | .WAV, .MP3 | 
| text file | .TXT |
| image file | .PNG, .JPG |
| video file | .MP4 |
| numerical data / categorical data | .JSON, .CSV |
